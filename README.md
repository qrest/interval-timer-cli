> interval timer for the cli


## Install

```sh
$ npm install --save node-interval-timer
```


## Usage

```js
var nodeIntervalTimer = require('node-interval-timer');

nodeIntervalTimer('Rainbow');
```

```sh
$ npm install --global node-interval-timer
$ node-interval-timer --help
```
